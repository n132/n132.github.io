cmp.py
```python
from pwn import *
a=ELF('/usr/max/lib/libc.so.6')
b=ELF('/lib/x86_64-linux-gnu/libc.so.6')
print hex(a.symbols['system'])
print hex(b.symbols['system'])
```

result：
```
0x3f830
0x42510
```
