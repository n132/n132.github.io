# int 0x80
这个也需要凑..
因为我们有
```python
[26]     xor dh,BYTE PTR [esi]
[07]     xor BYTE PTR [edi],dh
```
所以我们先set esi edi=0方法同第一次
```asm
push esi
dec eax
dec eax
dec eax
dec eax
xor esi,DWORD PTR [eax]
```
然后set esi edi 为我们可以控制的地方例如0x324f6f43...
然后我们利用`26`,`07`...
所以我们又开始凑
A:0x?? B:0x?? C:0x88
目标是A^B^C==0xcd和0x80
因为第二个字节受前一个字节的约束所以我们先弄第二个字节
## 0x80
0x80比较简单AB只要凑出8就可以了
我们可以随便选择像是
